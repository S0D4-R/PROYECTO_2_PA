
# Proyecto 02 "BARBERIA"

El presente programa fue desarrollado con el propósito de gestionar de forma eficiente las operaciones principales de una barbería, incluyendo la administración de servicios, citas y cobros. Está construido utilizando el lenguaje de programación Python, junto con la biblioteca Tkinter para el diseño de interfaces gráficas interactivas, lo que permite una experiencia visual intuitiva y fácil de usar.

El sistema se conecta directamente a una base de datos, desde la cual obtiene y organiza la información necesaria para cada módulo. Gracias a su estructura modular, el programa permite realizar acciones como registrar, visualizar, editar o eliminar datos de manera ordenada. Además, implementa principios de programación estructurada, utilizando iteraciones y referencias a objetos (punteros) para optimizar el flujo de información dentro del sistema.

En conjunto, el proyecto representa una solución práctica que integra el manejo de datos con una interfaz amigable, facilitando el control administrativo dentro de un entorno profesional.
